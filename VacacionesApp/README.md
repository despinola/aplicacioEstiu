# 🏖️ Vacaciones de Agosto - App Familiar

Aplicación Android para seguimiento de actividades y juegos diarios durante las vacaciones de agosto.

## 📱 Características

- **4 usuarios**: Blai, Rita, Miriam y David
- **31 juegos diferentes**: Uno para cada día de agosto
- **Registro diario de actividades**:
  - 📖 Lectura
  - 🪥 Cepillado de dientes
  - 📺 Ver televisión
  - 🏊 Piscina
  - ✍️ Notas del día
- **Seguimiento de progreso**: Cada usuario tiene su propio progreso guardado

## 🎮 Juegos incluidos

1. Adivinar banderas de países
2. Sumas y restas
3. Capitales de Europa
4. Snake (serpiente)
5. Memory de animales
6. Multiplicaciones
7. Banderas de América
8. Sopa de letras
9. Ahorcado
10. Capitales del mundo
... y 21 más!

## 🛠️ Tecnologías

- **Kotlin** - Lenguaje de programación
- **Jetpack Compose** - UI moderna y declarativa
- **Room Database** - Persistencia de datos local
- **Material Design 3** - Sistema de diseño

## 🚀 Cómo ejecutar

1. Abre el proyecto en Android Studio
2. Espera a que Gradle sincronice
3. Conecta un dispositivo Android o inicia un emulador
4. Pulsa Run ▶️

## 🎯 Requisitos

- Android Studio Arctic Fox o superior
- Android SDK 24+ (Android 7.0 Nougat)
- Kotlin 1.9.0+
